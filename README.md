# The Font App

(This is the currently maintained web version of what once was a [desktop app](https://github.com/andrwui/the-font-app-desktop.git))

## Not currently maintained
This project is not (currently) being mantained.
It has a lot of known issues, but I think that right now I should move on with some other projects and learn new things :) 

## Current features

- Preview, find, and tweak the fonts installed on your computer
- Google fonts previewer
- Preview local font Glyphs
- Add items to collections (Local fonts currently broken)
